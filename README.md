# XiwA-v2

Bienvenue dans **XiwA-v2**, un vrai template inspiré par l'esthétique anime la plus sombre. Ici, chaque ligne de code transpire la nuit, l'asphalte mouillé sous les néons, et la solitude d'une ville qui ne dort jamais.

---

## Présentation

XiwA-v2 est conçu pour ceux qui n'ont pas peur du noir :
- Palette de couleurs noire profonde, prête à engloutir la lumière
- Prise en charge du mode nuit uniquement, aucune lueur ne s'échappe
- Architecture inspirée des univers cyberpunk et anime noirs (dystopie, fatalisme, mystère)
- Code modulaire, structuré pour la lisibilité même dans les ténèbres

---

## Installation

Clone le dépôt et installe les dépendances, dans l'ombre de ton terminal :

```bash
git clone https://github.com/ton-ombre/XiwA-v2.git
cd XiwA-v2
npm install
```
